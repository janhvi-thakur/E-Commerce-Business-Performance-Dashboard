# 📊 E-Commerce Business Performance Dashboard

## Project Overview
This project analyzes e-commerce sales data using Python and Power BI to identify trends, customer behavior, and business opportunities.

## Objectives
- Analyze sales performance
- Identify top-performing categories and regions
- Study customer segments
- Build an executive dashboard

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Power BI
- DAX

## Key KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Sales

## Business Insights
- Technology generated the highest sales.
- The West region contributed the most revenue.
- Consumer customers represented the largest sales segment.
- Home Office has growth potential.
